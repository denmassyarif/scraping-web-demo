# Scraping Web Demo

This project is a simple web demo that displays structured product data collected using a Python web scraping script.

The scraping process extracts product details such as SKU, product name, brand, specifications, and images, then exports the data into a CSV file which is displayed in this web interface.

---

## Tech Stack
- HTML5
- CSS3
- Python (Requests, BeautifulSoup, Pandas)
